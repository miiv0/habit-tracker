
# Habit Tracker

A simple and intuitive habit tracking application to help you build and maintain positive routines.

## Features

- **Create Tasks** - Add new habits to track your daily progress
- **Recurring Tasks** - Set tasks to repeat on a custom schedule
- **Color Customization** - Assign colors to tasks for better organization and visibility
- **Remove Tasks** - Delete habits you no longer need to track
- **Light/Dark Mode** - Toggle between light and dark themes for comfortable viewing

## Getting Started

1. Launch the application
2. Click "Create Task" to add your first habit
3. Configure frequency and color settings
4. Toggle dark mode in settings as needed

## Usage Tips

- Use different colors to categorize habits by type
- Set recurring schedules that match your routine
- Remove completed or unwanted habits anytime
